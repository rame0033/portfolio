Honey
Lyric video

With the release of "honey," a song by rienne, is the release also of its lyric video. The first thing they told me is that the character sketch is provided. It is hand drawn by rienne themselves.

I placed the character to Adobe Animate to set the rigging. This is also my first time to use the software.

Following the vibe, the character needs to be in an environment so I set the background to a street sidewalk and trees so that it wouldn't be busy when the lyrics are being shown.I also set the lyrics to Grimnotes font to match the overall look. Included a white outline on text since there are times that it will be unread because it overlaps the background.

The environment, and lyric sync were done and polished in After Effects. See the video below in action.
