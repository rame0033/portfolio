Unit Converter

Go to Site Go to Github

This project is part of the practice time of the "Learn JavaScript" module of Scrimba

The converter is being built from scratch and the display layout is based from the Figma file provided on the lesson.

The input value will be multiplied to the corresponding value of the respective unit. The target unit are as follows:

  • Meter to Feet (1m = 3.281ft) and vice versa (1ft = 0.3048m)
  • Liter to Gallon (1L = 0.264gal) and vice versa (1gal = 3.785L)
  • Kilogram to Pound (1kg = 2.205lb) and vice versa (1lb = 0.4536kg)
